# バトルフェス開発裏話　その5

今回は、バトルフェスの「メインロゴ」についてです。

実は、現在のロゴに決まるまでに、いくつもの案を作って比較しました。

「夏フェスらしい楽しさ」を強くしたものや、「チーム同士が競うバトル感」を前面に出したもの。

ただ、ロゴ単体で格好いいかどうかだけでは決められませんでした。

実際に特設サイトへ置いた時に、背景やフォント、花火やゴールドの光と合うか。サイト全体のイメージから浮いてしまわないか。そこもかなり吟味しました。

派手にしすぎると文字が読みにくくなり、見やすさを優先すると、今度はイベントの熱量が足りない。

そのバランスに悩みながら、最終的には、イベント名がひと目で伝わり、「夏フェスの熱気」と「勝負が始まる高揚感」の両方を感じられるデザインを選びました。

今回は、比較できるように、これまで作ったメインロゴを全部並べてみます！

皆さんは、どのロゴが一番好きですか？
